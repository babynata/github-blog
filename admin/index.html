Publish Steps

发文前的最短路径

  1. 复制 `content/posts/_template.md` 新建文章。
  2. 填写标题、slug、日期、摘要、标签和正文。
  3. 运行 `node scripts/build-posts.mjs` 生成页面。
  4. 提交并 push 到 GitHub，Pages 会自动更新。
node scripts/build-posts.mjs

你也可以用 `npm run build:posts`。`draft` 状态不会出现在首页，也不会生成公开文章页。
