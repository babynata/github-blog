About The Layout

为长期写作留出呼吸感。

页面刻意减少了视觉噪声，把注意力集中在标题、摘要和正文节奏上。 绿色被压低了饱和度，不会显得冷，也不会显得过于情绪化。

  • 纯静态结构，天然适合 GitHub Pages
  • 文章源改为 `content/posts/*.md`，不用再手写 HTML
  • 首页展示文章列表，单篇文章自动生成独立页面
  • 标签筛选、轻动画、内容台和响应式布局都已做好
  • 运行一次构建脚本，再 push 到 GitHub 就能上线

小而明确的碎片

背景

没有用纯色铺满，而是叠了淡绿色渐变和细网格，让页面看起来更柔和。

02

动效

只保留了进入时的轻微上浮和卡片悬停反馈，避免博客显得像展示型 landing page。

03

排版
